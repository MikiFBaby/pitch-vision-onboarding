## How to Apply

1. **Identify the task type** - What specialized expertise is needed?
2. **Select the agent** - Match the task to the right persona
3. **Adopt the mindset** - Apply that agent's:
   - Personality traits
   - Critical rules
   - Workflow process
   - Deliverable standards
4. **Measure success** - Use the agent's success metrics
